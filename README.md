#### Установка и настройка
>
> Склонировать репозиторий и перейти в каталог 

```bash
make db_setup 		# Зависимости
make db_migrate 	# Миграции базы
make db_seed 		# Тестовый набор данных. Не обязательно
```

# Поля

- Категория
- Комментарий
- Время ухода
- Время прихода
